# \# AGENTS.md - AI训练工具项目设计规范

# 

# \## 强制设计规则（禁止违反）

# 

# \### 配色系统（GitHub Dark 风格）

# \- 主背景: #0d1117

# \- 次级背景: #161b22

# \- 边框: #21262d

# \- 主文字: #c9d1d9

# \- 次文字: #8b949e

# \- 强调蓝: #58a6ff

# \- 训练绿: #3fb950

# \- 危险红: #f85149

# 

# \### 禁止事项

# \- ❌ 禁止使用系统默认配色和默认字体

# \- ❌ 禁止使用蓝紫渐变

# \- ❌ 禁止使用 Emoji 作为图标

# \- ❌ 禁止使用纯白背景

# \- ❌ 禁止使用 "Label1" "Button" 等占位符文案

# \- ❌ 禁止在业务代码中硬编码颜色（走QSS）

# \- ❌ 禁止使用 matplotlib 嵌入（用QPainter自绘）

# 

# \### 字体

# \- 全局: "JetBrains Mono", "Consolas", monospace

# \- 字号: 11px(说明) / 12px(正文) / 13px(默认) / 15px(标题) / 20px(指标)

# 

# \### 布局

# \- 侧边导航: 200px 固定宽度

# \- 内容区内边距: 24px

# \- 控件间距: 12px / 16px / 20px（只允许这三个值）

# \- 圆角: 卡片8px / 按钮6px / 进度条3px

# 

# \### 组件规范

# \- 按钮: 普通(#21262d) / 主按钮(#238636) / 危险(#da3633)

# \- 指标卡片: 80px高，上方11px大写灰色标签，下方20px绿色粗体数值

# \- 表格: 表头11px大写，行高36px

# \- 日志: QPlainTextEdit，等宽字体11px
